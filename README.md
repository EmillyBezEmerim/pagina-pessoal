# pagina-pessoal
Página pessoal desenvolvida com HTML5 e CSS3, com o objetivo de apresentar informações profissionais, conhecimentos técnicos e links relevantes.

## Tecnologias utilizadas
- HTML5
- CSS3

## Objetivo
Projeto criado para aprendizado e prática de desenvolvimento front-end, além de servir como portfólio inicial na área de Tecnologia da Informação.
